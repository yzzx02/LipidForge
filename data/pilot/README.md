# Pilot Data

`experimental_ms2_pilot.jsonl` contains 6 MassBank-derived experimental MS/MS
records used only for pipeline validation.

Use this file for smoke tests, parser tests, model forward checks, loss checks,
backward checks, and small overfit tests. Do not report model performance from
this pilot set.

The larger raw data package and LIPID MAPS structure seed files are not included
in this repository.
